\e[0;1mUsage\e[0m:
	Launch editor:
		mcide [options...] FILES...
	Get help:
		mcide <-h|--help> [<--lang|--language|-l> lang:LANG]
	Recover Terminal:
		# If the game crashed, it may affect your terminal, then run this:
		mcide <--rt|--recover-terminal>
\e[1mOptions:\e[0m
	\e[4mOptions with \e[31mred\e[0;4m color IS NOT AVAILBLE now\e[0m
	\e[4mOptions with \e[33myellow\e[0;4m color HAVE NO EFFECT now\e[0m
		--dir dir:PATH		use $dir as the game dir
		--menu, -m		show game menu(page=menu)
		\e[31m--create-world		create a new world(page=create_world)\e[0m
		--load-world file:PATH,	load a world
		\e[37m|\e[0m --open-world file:PATH,
		\e[37m|\e[0m -w file:PATH
		\e[33m--language lang:LANG	set the language
		\e[37m|\e[0m \e[33m--lang lang:LANG, -l lang:LANG\e[0m
		--vision-size x:NUMBER y:NUMBER	Set the vision size(render distance)
		--vision-size-x x:NUMBER	Set the vision width
		\e[37m|\e[0m -x x:NUMBER
		--vision-size-y y:NUMBER	Set the vision height
		\e[37m|\e[0m -y y:NUMBER
	* If any dimnesion is defined in the arguments, the default page
	*  will be setted to `create_world`, otherwise it will be setted
	*  to `menu`.
	\e[1m For `menu` page: \e[0m
	\e[1m For `create_world` page: \e[0m
		--dim dimid:DID file:PATH	read a dimension from a file, \e[31mleave $dimid '-' to auto set\e[0m
		\e[37m|\e[0m --dimension dimid:DID file:PATH,
		\e[37m|\e[0m -d dimid:DID file:PATH
		--overworld file:PATH		read overworld(dim `mcide:overworld`) from a file
		\e[37m|\e[0m -o file:PATH
	\e[1m For `load world`: \e[0m
